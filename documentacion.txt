ATAJOS EXCEL
------------

CTRL + ESPACIO: Selecciona toda la columna
SHIFT + ESPACIO: Selecciona toda la fila

CTRL + : Agregar una fila
CTRL - : Eliminar una fila

CTRL + FLECHA ARRIBA: Sube hasta la ultima celda con valor
CTRL + FLECHA ABAJO: Baja hasta la ultima celda con valor
CTRL + FLECHA IZQUIERDA: Ultima celda con valor a la izquierda
CTRL + FLECHA DERECHA: Ultima celda con valor a la derecha
CTRL + 0: Oculta la columna
CTRL + U: Crear nueva hoja en un nuevo archivo excel

Ctrl + Fn + RePag - Moverse entre hojas
Crtl + Fn + AvPag - Moverse entre hojas

LISTAS
ALT + : Para ver opciones de la lista



CTRL + 1: Abre el formato de celdas
ALT + F11: Abre el editor VBA
CTRL + “-”: Elimina toda la fila o columna en caso de haber utilizado 
los dos anteriores


CTRL + SHIFT + L: Activa/Desactiva los filtros
CTRL + 9: Oculta la fila
CTRL + SHIFT + 0: Muestra la columna
CTRL + SHIFT + 9: Muestra la fila
